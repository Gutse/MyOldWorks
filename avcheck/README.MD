Утилита предназначена для автоматический установки различного ПО, с возможностью автоматического выбора конкретного ПО из категории в зависимости от конфигурации железа. 
Задумывалось для автоматический установки антивируса, на старом железе новый антивирус не работал, для старых пк устанавливался нод, для более мощных ставился symantec, который в последствии был заменен на kav.
Прописывается в Logon скрипт для всех пользователей.

Так же была добавлена функция добавления записи в базу данных ПК у пользователей (см утилиту domain_lookup). Суть такова - при успешном логине пользователя на пк, программа сообщает на сервер следующую информацию:
текущий пользватель, IP, имя компьютера.
