Небольшая утилита, разработана для анализа файлов с данными о проходах работников черех проходные полученных из СКУД (система контроля и управления доступом) Perco, с дальнейшим их преобразованием в требуемый начальством формат.
Понадобилась как замена модулю учета рабочего времени.

В файле ThisOutlookSession.cls находится модуль для Outlook для автоматизации обработки таких файлов. 

Файлы с данными поступают от работника службы безопасности. В Outlook нужно настроить правило - при получении писем с вложениями от этого сотрудника - запустить этот модуль.
Модуль, для каждого подходящего вложения вызовет программу конвертации и полученный результат отправит начальству.
